데이터 가져오기 사용 가이드

사용자 버전
▾

로컬 네트워크를 통해 컴퓨터의 데이터 파일을 TimeTrails로 가져올 수 있습니다.

전체 과정 데이터 케이블이 필요 없습니다. 도 로그인 계정이 필요하지 않습니다. .

1. 준비 작업

✓
휴대폰과 컴퓨터가 같은 Wi-Fi에 연결되어 있습니다
✓
가져올 파일을 준비하세요
데이터베이스 파일 .db 및 트랙 파일 .gpx을(를) 지원합니다.

2. 모바일에서 가져오기 서비스 활성화

  1. 1
    TimeTrails를 엽니다
  2. 2
    「데이터 가져오기 / 데이터 전송」 페이지로 이동
  3. 3
    서비스 시작 클릭
  4. 4
    성공하면 페이지에 URL(로컬 네트워크 주소)이 표시됩니다.
    예: http://192.168.1.100:9527

셋, 컴퓨터에서 파일 업로드하기

  1. 1
    컴퓨터 브라우저에 휴대폰에 표시된 그 URL을 입력하세요.
  2. 2
    열고 파일을 선택하여 업로드하세요.
  3. 3
    업로드가 완료되면 파일이 자동으로 휴대폰의 파일 목록에 나타납니다.

4. 데이터 가져오기

휴대폰의 파일 목록에서:

.db 데이터베이스 파일

전체 TimeTrails 데이터를 복구하거나 이전하는 데 사용됩니다.

  1. 1
    .db 파일을 클릭하세요.
  2. 2
    "가져오기" 선택
  3. 3
    완료 후 성공 메시지가 표시됩니다.

.gpx 경로 파일

가져오기가 완료되면 날짜별로 지도에 표시됩니다.

  1. 1
    클릭하여 .gpx 파일
  2. 2
    시스템이 자동으로 경로를 분석하고 가져옵니다.
  3. 3
    완료 후 지도에서 확인하세요
⚠️ 가져오는 동안 앱을 포그라운드에 유지해 주세요. 완료되면 성공 메시지를 볼 수 있습니다.

5. 일반적인 팁 설명

!
서비스를 시작할 수 없습니다
현재 연결된 네트워크가 셀룰러 네트워크가 아닌 Wi-Fi인지 확인하십시오.
!
가져오기 실패 또는 일부 데이터가 건너뛰어짐
일부 트랙 포인트에 시간 정보가 누락되어 있어 시스템이 자동으로 무시합니다.
!
파일을 찾을 수 없습니다.
드롭다운 목록을 새로 고침하면 다시 스캔할 수 있습니다.

6. 작은 제안

✓
가져오기 전에 현재 데이터를 백업하는 것이 좋습니다.
✓
.db 파일은 일반적으로 전체 마이그레이션에 사용됩니다.
✓
.gpx 단일 또는 다중 경로 기록을 가져오는 데 더 적합합니다.
