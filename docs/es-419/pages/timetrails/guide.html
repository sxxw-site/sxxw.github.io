Guía del Usuario para la Importación de Datos

Versión de Usuario
▾

Puedes importar archivos de datos desde tu computadora a TimeTrails a través de la red de área local.

Todo el proceso No se necesita cable de datos , también No se requiere inicio de sesión en la cuenta .

1. Trabajo de Preparación

✓
Asegúrate de que <b> móvil y computadora estén conectados a la misma red Wi-Fi </b>.
✓
Prepara el archivo para ser importado
Soporta el archivo de base de datos <code class="code-blue">.db</code> y el archivo de pista <code class="code-green">.gpx</code>

2. Habilitar el Servicio de Importación en Móvil

  1. 1
    Abre TimeTrails
  2. 2
    Ingrese a la página 'Importación de Datos / Transferencia de Datos'
  3. 3
    Haz clic en 'Iniciar Servicio'
  4. 4
    Después de tener éxito, se mostrará una URL (dirección de red local) en la página.
    Por ejemplo: http://192.168.1.100:9527

3. Cargando Archivos en la Computadora

  1. 1
    En el navegador de tu computadora, ingresa la URL que se muestra en tu teléfono.
  2. 2
    Ábrelo, luego selecciona y sube tu archivo.
  3. 3
    Una vez que la carga esté completa, el archivo aparecerá automáticamente en la lista de archivos en tu teléfono.

4. Importar Datos

En la lista de archivos en tu teléfono:

archivo de base de datos <code class="code-blue">.db</code>

Se utiliza para restaurar o migrar todos los datos de TimeTrails

  1. 1
    Haz clic en el archivo .db
  2. 2
    Selecciona 'Importar'
  3. 3
    Se mostrará un mensaje de éxito al completar.

archivo de pista <code class="code-green">.gpx</code>

Mostrado en el mapa por fecha después de que la importación esté completa.

  1. 1
    Haz clic en el archivo .gpx
  2. 2
    El sistema analizará e importará automáticamente la ruta.
  3. 3
    Revisa el mapa después de completar
⚠️ Por favor, mantén la App en primer plano durante el proceso de importación, y verás un mensaje de éxito una vez completado.

5. Consejos y Explicaciones Comunes

!
No se puede iniciar el servicio
Por favor, asegúrate de que estás conectado a Wi-Fi, no a la red celular.
!
La importación falló o se omitieron algunos datos
Algunos puntos de la pista faltan información de tiempo, y el sistema los ignorará automáticamente.
!
Archivo no encontrado
Actualiza la lista desplegable para volver a escanear.

6. Algunas Sugerencias

✓
Antes de importar, se recomienda hacer una copia de seguridad de los datos actuales.
✓
Los archivos <code class="code-blue">.db</code> se utilizan típicamente para migraciones completas.
✓
<code class="code-green">.gpx</code> es más adecuado para importar registros de pista individuales o múltiples.
