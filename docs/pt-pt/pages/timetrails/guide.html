Guia do Utilizador para Importação de Dados

Versão do Utilizador

Pode importar ficheiros de dados do seu computador para o TimeTrails através da rede local.

Todo o processo Não é necessário cabo de dados , também Não é necessário fazer login na conta .

1. Trabalho de Preparação

Certifique-se de que <b> o telemóvel e o computador estão ligados à mesma rede Wi-Fi </b>.
Prepare o ficheiro a ser importado
Suporta o ficheiro de base de dados <code class="code-blue">.db</code> e o ficheiro de pista <code class="code-green">.gpx</code>

2. Ativar o Serviço de Importação no Móvel

  1. 1Abra o TimeTrails
  2. 2Aceda à página 'Importação de Dados / Transferência de Dados'
  3. 3Clique em 'Iniciar Serviço'
  4. 4
    Após o sucesso, um URL (endereço de rede local) será exibido na página.
    Por exemplo: http://192.168.1.100:9527

3. Carregar Ficheiros no Computador

  1. 1No navegador do seu computador, insira a URL exibida no seu telefone.
  2. 2Abra-o, depois selecione e carregue o seu ficheiro.
  3. 3Após o upload ser concluído, o ficheiro aparecerá automaticamente na lista de ficheiros no seu telefone.

4. Importar Dados

Na lista de ficheiros no seu telefone:

ficheiro de base de dados <code class="code-blue">.db</code>

Usado para restaurar ou migrar todos os dados do TimeTrails

1Clique no ficheiro .db
2Selecione 'Importar'
3Uma mensagem de sucesso será exibida após a conclusão.

arquivo de track <code class="code-green">.gpx</code>

Exibido no mapa por data após a importação estar completa.

1Clique no ficheiro .gpx
2O sistema irá analisar e importar automaticamente a trilha.
3Verifique o mapa após a conclusão
⚠️ Por favor, mantenha a App em primeiro plano durante o processo de importação, e verá uma mensagem de sucesso assim que estiver concluído.

5. Dicas e Explicações Comuns

Incapaz de iniciar o serviço

Por favor, assegure-se de que está conectado ao Wi-Fi, e não à rede celular.

Importação falhou ou alguns dados foram ignorados

Alguns pontos de pista estão a faltar informações de tempo, e o sistema irá ignorá-los automaticamente.

Ficheiro não encontrado

Atualize a lista suspensa para reanalisar.
