Data Import User Guide

User Version
▾

You can import data files from your computer into TimeTrails via the local area network.

The whole process No data cable needed , also No account login required .

1. Preparation Work

✓
Connect your phone and computer to the same Wi-Fi
✓
Prepare the file to be imported
Supports database file .db and track file .gpx

2. Enable Import Service on Mobile

  1. 1
    Open TimeTrails
  2. 2
    Enter the 'Data Import / Data Transfer' page
  3. 3
    Click 'Start Service'
  4. 4
    After success, a URL (local network address) will be displayed on the page.
    For example: http://192.168.1.100:9527

3. Uploading Files on the Computer

  1. 1
    In your computer's browser, enter the URL displayed on your phone.
  2. 2
    Open it, then select and upload your file.
  3. 3
    After the upload is complete, the file will automatically appear in the file list on your phone.

4. Import Data

In the file list on your phone:

.db Database File

Used to restore or migrate entire TimeTrails data

  1. 1
    Click the .db file
  2. 2
    Select 'Import'
  3. 3
    A success message will be displayed upon completion.

.gpx Track File

Displayed on the map by date after the import is complete.

  1. 1
    Click the .gpx file
  2. 2
    The system will automatically parse and import the track.
  3. 3
    Check the map after completion
⚠️ Please keep the App in the foreground during the import process, and you will see a success message once completed.

5. Common Tips and Explanations

!
Unable to start the service
Please ensure that you are connected to Wi-Fi, not cellular network.
!
Import failed or some data was skipped
Some track points are missing time information, and the system will automatically ignore them.
!
File not found
Refresh the dropdown list to rescan.

6. Some Suggestions

✓
Before importing, it is recommended to back up the current data.
✓
.db Files are typically used for complete migration.
✓
.gpx More suitable for importing single or multiple track records
