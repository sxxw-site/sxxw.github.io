Руководство пользователя по импорту данных

Версия пользователя
▾

Вы можете импортировать файлы данных с вашего компьютера в TimeTrails через локальную сеть.

Весь процесс Кабель данных не нужен , также Не требуется вход в аккаунт .

1. Подготовительная работа

✓
Подключите свой телефон и компьютер к одной и той же сети Wi-Fi
✓
Подготовьте файл для импорта
Поддерживает файл базы данных .db и файл трека .gpx

2. Включите службу импорта на мобильном устройстве

  1. 1
    Откройте TimeTrails
  2. 2
    Перейдите на страницу 'Импорт данных / Передача данных'
  3. 3
    Нажмите 'Начать сервис'
  4. 4
    После успешного завершения на странице будет отображен URL (адрес локальной сети).
    Например: http://192.168.1.100:9527

3. Загрузка файлов на компьютер

  1. 1
    В браузере вашего компьютера введите URL, отображаемый на вашем телефоне.
  2. 2
    Откройте его, затем выберите и загрузите ваш файл.
  3. 3
    После завершения загрузки файл автоматически появится в списке файлов на вашем телефоне.

4. Импорт данных

В списке файлов на вашем телефоне:

.db Файл базы данных

Используется для восстановления или миграции всех данных TimeTrails

  1. 1
    Нажмите на файл .db
  2. 2
    Выберите 'Импорт'
  3. 3
    Сообщение об успешном завершении будет отображено по окончании.

.gpx Файл трека

Отображается на карте по дате после завершения импорта.

  1. 1
    Нажмите на файл .gpx
  2. 2
    Система автоматически разберет и импортирует трек.
  3. 3
    Проверьте карту после завершения
⚠️ Пожалуйста, держите приложение на переднем плане во время процесса импорта, и вы увидите сообщение об успешном завершении, как только он будет завершен.

5. Общие советы и объяснения

!
Не удается запустить службу
Пожалуйста, убедитесь, что вы подключены к Wi-Fi, а не к сотовой сети.
!
Импорт не удался или некоторые данные были пропущены
Некоторые точки трека не содержат информации о времени, и система автоматически проигнорирует их.
!
Файл не найден
Обновите выпадающий список для повторного сканирования.

6. Некоторые предложения

✓
Перед импортом рекомендуется создать резервную копию текущих данных.
✓
.db Файлы обычно используются для полной миграции.
✓
.gpx Более подходит для импорта одной или нескольких записей трека
