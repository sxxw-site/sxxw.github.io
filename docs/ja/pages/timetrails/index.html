時光軌跡 TimeTrails

携帯電話の「どこに行ったか、どれくらい滞在したか、どれだけの距離を移動したか」を振り返ることができる個人的なタイムラインに変えます。

あなたの足跡は、「受動的にクラウドに上がる一連の点」だけではないはずです

日常の軌跡を記録する価値は、「たくさんのポイント」ではなく、振り返り、整理し、復習できることです。TimeTrailsはあなたの移動を明確なタイムラインに変えます：今日はどこに行き、どれくらい滞在し、どれだけ歩いたのか、一目で理解できます。同時に、私たちはローカルファーストとデータ収集の最小化を貫いています——あなたが記録するのは、自分を理解するためであり、プラットフォームに理解されるためではありません。

デフォルトではローカルにのみ保存されます。同期、エクスポート、誰に見せるかはすべてあなたが決めます。

それを使って何をしますか

  • 毎日自動生成される「今日のトレイル」：ルート、距離、滞在ポイントが一目でわかる
  • 過去の旅行を振り返る：日別/場所別にあなたのアクションタイムラインを閲覧する
  • 個人の振り返り：通勤、運動、移動パターンの長期的な変化
  • ある日を思い出す：タイムラインで「その店」「その道」「その散歩」を素早く見つける
  • あなたのペースを統計する：通勤/運動/旅行の距離のトレンド、長期的な比較がより直感的に

自動記録

できるだけ少ない干渉で、移動プロセスを軌跡と統計として記録します；権限はいつでも撤回可能です。

ビジュアル整理

タイムライン + マップの二重視点：日ごとに振り返り、場所ごとに集約し、距離ごとに振り返る。

プライバシー優先

デフォルトではローカルにのみ保存されます。いつでもクリア、エクスポート、または権限をオフにできます；不必要なデータ収集は行いません。

コアコンピタンス

複雑な機能を詰め込むのではなく、「トラッキング記録」ということを使いやすく、明確に、制御可能にすることに集中します。
