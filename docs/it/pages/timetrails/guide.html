Guida all'Importazione dei Dati

Versione Utente
▾

Puoi importare file di dati dal tuo computer in TimeTrails tramite la rete locale.

L'intero processo Nessun cavo dati necessario , anche Nessun accesso all'account richiesto .

1. Lavoro di Preparazione

✓
Assicurati che <b> mobile e computer siano connessi alla stessa rete Wi-Fi </b>.
✓
Prepara il file da importare
Supporta il file del database <code class="code-blue">.db</code> e il file del tracciato <code class="code-green">.gpx</code>

2. Abilita il Servizio di Importazione su Mobile

  1. 1
    Apri TimeTrails
  2. 2
    Accedi alla pagina 'Importazione Dati / Trasferimento Dati'
  3. 3
    Clicca su 'Inizia Servizio'
  4. 4
    Dopo il successo, un URL (indirizzo di rete locale) verrà visualizzato sulla pagina.
    Ad esempio: http://192.168.1.100:9527

3. Caricamento di file sul computer

  1. 1
    Nel browser del tuo computer, inserisci l'URL visualizzato sul tuo telefono.
  2. 2
    Aprilo, quindi seleziona e carica il tuo file.
  3. 3
    Dopo che il caricamento è completato, il file apparirà automaticamente nell'elenco dei file sul tuo telefono.

4. Importa Dati

Nell'elenco dei file sul tuo telefono:

file del database <code class="code-blue">.db</code>

Utilizzato per ripristinare o migrare l'intero database di TimeTrails

  1. 1
    Fai clic sul file .db
  2. 2
    Seleziona 'Importa'
  3. 3
    Un messaggio di successo verrà visualizzato al termine.

file di tracciamento <code class="code-green">.gpx</code>

Visualizzato sulla mappa per data dopo il completamento dell'importazione.

  1. 1
    Clicca sul file .gpx
  2. 2
    Il sistema analizzerà e importerà automaticamente il tracciato.
  3. 3
    Controlla la mappa dopo il completamento
⚠️ Si prega di mantenere l'App in primo piano durante il processo di importazione e vedrai un messaggio di successo una volta completato.

5. Suggerimenti e spiegazioni comuni

!
Impossibile avviare il servizio
Si prega di assicurarsi di essere connessi a Wi-Fi, non alla rete cellulare.
!
Importazione non riuscita o alcuni dati sono stati saltati
Alcuni punti del tracciato mancano di informazioni temporali e il sistema li ignorerà automaticamente.
!
File non trovato
Aggiorna l'elenco a discesa per riscanalizzare.

6. Alcuni Suggerimenti

✓
Prima di importare, si consiglia di eseguire il backup dei dati attuali.
✓
I file <code class="code-blue">.db</code> sono tipicamente utilizzati per la migrazione completa.
✓
<code class="code-green">.gpx</code> è più adatto per importare singoli o più registrazioni di tracce.
