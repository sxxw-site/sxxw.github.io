Användarguide för dataimport

Användarversion
▾

Du kan importera datafiler från din dator till TimeTrails via det lokala nätverket.

Hela processen Ingen datakabel behövs , också Ingen kontoinloggning krävs .

1. Förberedelsearbete

✓
Se till att <b> mobil och dator är anslutna till samma Wi-Fi </b>.
✓
Förbered filen som ska importeras
Stöder databasfil <code class="code-blue">.db</code> och spårfil <code class="code-green">.gpx</code>

2. Aktivera importtjänst på mobil

  1. 1
    Öppna TimeTrails
  2. 2
    Gå till sidan 'Data Import / Data Transfer'
  3. 3
    Klicka på 'Starta tjänst'
  4. 4
    Efter framgång kommer en URL (lokal nätverksadress) att visas på sidan.
    Till exempel: http://192.168.1.100:9527

3. Ladda upp filer på datorn

  1. 1
    I din dators webbläsare, ange URL:en som visas på din telefon.
  2. 2
    Öppna det, välj sedan och ladda upp din fil.
  3. 3
    När uppladdningen är klar kommer filen automatiskt att visas i fillistan på din telefon.

4. Importera data

I fillistan på din telefon:

<code class="code-blue">.db</code> databasfil

Används för att återställa eller migrera hela TimeTrails-data

  1. 1
    Klicka på .db-filen
  2. 2
    Välj 'Importera'
  3. 3
    Ett meddelande om framgång kommer att visas vid slutförandet.

<code class="code-green">.gpx</code> spårfil

Visas på kartan efter datum när importen är klar.

  1. 1
    Klicka på .gpx-filen
  2. 2
    Systemet kommer automatiskt att analysera och importera spåret.
  3. 3
    Kontrollera kartan efter slutförande
⚠️ Vänligen håll appen i förgrunden under importprocessen, och du kommer att se ett meddelande om framgång när det är klart.

5. Vanliga tips och förklaringar

!
Kunde inte starta tjänsten
Vänligen se till att du är ansluten till Wi-Fi, inte mobilnät.
!
Importen misslyckades eller viss data hoppades över
Vissa spårpunkter saknar tidsinformation, och systemet kommer automatiskt att ignorera dem.
!
Fil hittades inte
Uppdatera rullgardinslistan för att skanna om.

6. Några förslag

✓
Innan importering rekommenderas det att säkerhetskopiera de aktuella uppgifterna.
✓
<code class="code-blue">.db</code> filer används vanligtvis för fullständig migrering.
✓
<code class="code-green">.gpx</code> är mer lämplig för att importera enstaka eller flera spårregister.
