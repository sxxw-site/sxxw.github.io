Guide de l'utilisateur pour l'importation de données

Version utilisateur
▾

Vous pouvez importer des fichiers de données depuis votre ordinateur dans TimeTrails via le réseau local.

L'ensemble du processus Aucun câble de données requis , aussi Aucun identifiant de compte requis .

1. Travaux de préparation

✓
Connectez votre téléphone et votre ordinateur au même Wi-Fi
✓
Préparez le fichier à importer
Prend en charge le fichier de base de données .db et le fichier de piste .gpx

2. Activer le service d'importation sur mobile

  1. 1
    Ouvrez TimeTrails
  2. 2
    Accédez à la page 'Importation de données / Transfert de données'
  3. 3
    Cliquez sur 'Démarrer le service'
  4. 4
    Après le succès, une URL (adresse du réseau local) sera affichée sur la page.
    Par exemple : http://192.168.1.100:9527

3. Téléversement de fichiers sur l'ordinateur

  1. 1
    Dans le navigateur de votre ordinateur, entrez l'URL affichée sur votre téléphone.
  2. 2
    Ouvrez-le, puis sélectionnez et téléchargez votre fichier.
  3. 3
    Une fois le téléchargement terminé, le fichier apparaîtra automatiquement dans la liste des fichiers sur votre téléphone.

4. Importer des données

Dans la liste de fichiers sur votre téléphone :

.db Fichier de base de données

Utilisé pour restaurer ou migrer l'ensemble des données de TimeTrails

  1. 1
    Cliquez sur le fichier .db
  2. 2
    Sélectionnez 'Importer'
  3. 3
    Un message de succès sera affiché à l'achèvement.

.gpx Fichier de piste

Affiché sur la carte par date après l'importation.

  1. 1
    Cliquez sur le fichier .gpx
  2. 2
    Le système analysera et importera automatiquement la piste.
  3. 3
    Vérifiez la carte après l'achèvement
⚠️ Veuillez garder l'application au premier plan pendant le processus d'importation, et vous verrez un message de succès une fois terminé.

5. Conseils et explications courants

!
Impossible de démarrer le service
Veuillez vous assurer que vous êtes connecté à un réseau Wi-Fi, et non à un réseau cellulaire.
!
Échec de l'importation ou certaines données ont été omises
Certains points de piste manquent d'informations temporelles, et le système les ignorera automatiquement.
!
Fichier non trouvé
Rafraîchissez la liste déroulante pour rescanner.

6. Quelques suggestions

✓
Avant d'importer, il est recommandé de sauvegarder les données actuelles.
✓
.db Les fichiers sont généralement utilisés pour une migration complète.
✓
.gpx Plus adapté pour l'importation d'un ou plusieurs enregistrements de piste
